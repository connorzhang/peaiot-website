#AI 核心概念知识体系

#1. 概述

本文档构建了 PeaBSS 系统中人工智能(AI)相关的核心概念知识体系，旨在提供结构清晰、层级分明的知识框架，确保 AI 功能与用户权限体系的紧密集成。

#2. 核心概念分类

#2.1 智能体（Agent）

定义：具有自主决策和执行能力的 AI 系统，能够感知环境、制定计划并执行操作。

类型：

  • 对话式智能体：通过自然语言与用户交互
  • 工具型智能体：调用各种工具完成特定任务
  • 监控型智能体：实时监测系统状态和数据异常

权限关联：智能体的操作范围严格限定在用户权限范围内，确保数据访问安全。

#2.2 工具（Tool）

定义：智能体可以调用的功能模块，用于执行特定任务。

分类：

  • 数据查询工具：获取设备、报警、因子等数据
  • 分析工具：进行趋势分析、健康评估、合规性检查等
  • 系统工具：管理系统配置、监控系统状态
  • 通知工具：发送短信、微信等通知

权限控制：

  • 设备相关工具：只能访问用户所属组织的设备
  • 系统级工具：仅对超级管理员开放

#2.3 技能（Skill）

定义：智能体具备的特定能力，是工具的组合和应用。

分类：

  • 基础技能：获取设备列表、实时数据、因子字典等
  • 分析技能：健康评分、污染趋势、合规报告等
  • 预测技能：数据预测、异常检测、环境影响评估等
  • 管理技能：系统监控、告警管理、配置管理等

权限关联：技能的使用范围与用户权限直接对应，确保功能访问与数据展示的一致性。

#2.4 多租户隔离

定义：在多用户环境中，确保不同用户间的数据和操作相互隔离。

技术实现：

  • 数据库层面：通过 org_id 字段过滤数据
  • 应用层面：用户认证和权限验证
  • 接口层面：权限检查和数据过滤

安全保障：防止越权访问和数据泄露，确保用户只能访问其权限范围内的资源。

#4. 权限体系集成

#4.1 功能关联范围

所有技能模块、工具应用及相关功能必须严格基于当前登录账号的权限体系，精准展示其名下已授权关联的设备数据及企业信息。

#4.2 内容过滤规则

系统级功能（包括但不限于系统健康状态监控、全局配置管理、跨账号数据统计等）严禁在账号关联功能区域展示，确保用户界面只呈现与当前账号权限直接相关的内容。

#4.3 数据隔离要求

实施严格的多租户数据隔离机制，明确区分不同账号间的数据边界，通过权限校验、数据过滤和访问控制等技术手段，确保用户只能访问和管理其权限范围内的设备与企业资源。

#5. 交互体验优化

#5.1 界面设计

  • 个人视角：展示用户个人权限范围内的设备和数据
  • 企业视角：展示用户所在组织的整体状态和数据
  • 权限标识：清晰标识当前用户的权限级别和可访问范围

#5.2 数据展示

  • 排序规则：按照预设排序因子进行优先级排列
  • 环境监测指标：严格依据排序字段值进行降序或升序排列展示
  • 数据可视化：根据用户权限动态调整可展示的数据范围

#6. 安全保障

#6.1 认证机制

  • JWT 令牌认证
  • 会话管理
  • 密码加密存储

#6.2 授权机制

  • 基于角色的访问控制 (RBAC)
  • 细粒度的权限管理
  • 操作审计日志

#6.3 数据安全

  • 传输加密
  • 存储加密
  • 数据脱敏

#7. 实现细节

#7.1 权限检查流程

  1. 用户登录获取认证令牌
  2. 访问 API 时验证令牌有效性
  3. 获取用户信息和权限级别
  4. 根据权限级别过滤数据访问
  5. 记录操作审计日志

#7.2 多租户数据隔离实现

  • 数据库查询时添加 org_id 条件
  • 设备操作时验证设备归属
  • 报警查询时过滤组织权限
  • 系统工具访问时检查管理员权限

#7.3 AI 功能权限控制

  • 智能体只能调用用户有权访问的工具
  • 对话内容基于用户权限范围生成
  • 分析结果仅包含用户可访问的数据
  • 预测模型使用用户权限范围内的训练数据

#8. 最佳实践

#8.1 权限设计

  • 遵循最小权限原则
  • 定期审查权限设置
  • 建立权限变更审批流程

#8.2 性能优化

  • 缓存权限检查结果
  • 优化数据库查询
  • 使用索引加速权限过滤

#8.3 安全审计

  • 记录所有权限变更
  • 监控异常访问模式
  • 定期安全评估

#9. 总结

本知识体系文档构建了 PeaBSS 系统中 AI 相关核心概念的完整框架，确保了 AI 功能与用户权限体系的紧密集成，实现了严格的多租户数据隔离，为系统的安全性和可靠性提供了有力保障。

通过本知识体系的实施，PeaBSS 系统能够在提供智能服务的同时，确保数据安全和权限控制，为用户提供既智能又安全的环保物联网解决方案。
