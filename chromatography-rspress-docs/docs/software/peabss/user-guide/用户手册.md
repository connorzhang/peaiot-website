# PeaBSS 用户手册

## 1. 系统概述

PeaBSS（豌豆易联业务服务支撑系统）是一个专业的环保物联网平台，用于环保监测设备的数据采集、存储、分析和可视化。本系统支持多种环保监测协议，提供实时数据监控、告警管理、数据分析和智能报表等功能。

### 1.1 系统特点

- **多协议支持**: 支持 HJ212、MQTT、Modbus 等多种设备协议
- **实时监控**: 实时采集和展示设备数据
- **智能分析**: 基于 AI 的数据分析和预测
- **地理信息集成**: 结合 GIS 地图展示设备分布和状态
- **多端访问**: 支持 Web 端和移动 APP 访问
- **告警管理**: 基于阈值的告警和多渠道通知

### 1.2 适用场景

- **环境监测**: 空气质量、水质、噪声等环境参数监测
- **工业环保**: 工厂废气、废水排放监测
- **城市环境**: 城市空气质量监测网络
- **园区监测**: 产业园区环境监测
- **建筑工地**: 扬尘、噪声监测

## 2. 快速入门

### 2.1 登录系统

1. **打开浏览器**，输入系统地址（默认：http://localhost:1870）
2. **输入用户名和密码**
   - 管理员账号: admin / Adm@#2022!
   - 普通用户账号: master / opr123
3. **点击登录**进入系统

### 2.2 系统界面

- **左侧菜单**: 系统功能导航
- **顶部导航**: 系统标题、用户信息、退出按钮
- **主内容区**: 功能模块内容
- **底部状态栏**: 系统版本、状态信息

### 2.3 首次使用

1. **初始化数据库**: 首次使用需要运行 `./peabss -initdb` 初始化数据库
2. **添加设备**: 在设备管理中添加监测设备
3. **配置告警**: 设置设备告警阈值
4. **查看数据**: 在数据监测中查看设备数据
5. **设置大屏**: 配置 GIS 大屏和 AI BI 大屏

## 3. 设备管理

### 3.1 设备列表

- **查看设备**: 在左侧菜单中点击「设备管理」→「设备列表」
- **搜索设备**: 使用搜索框按设备名称、MN 号等搜索设备
- **筛选设备**: 按组织、项目、状态等筛选设备
- **设备详情**: 点击设备名称查看详细信息

### 3.2 添加设备

1. **点击添加按钮**: 在设备列表页面点击「添加」按钮
2. **填写设备信息**:
   - 设备名称: 设备的显示名称
   - MN 号: 设备唯一标识符
   - 设备类型: 选择设备类型（如 peaedge、dtu 等）
   - 组织 ID: 选择设备所属组织
   - 项目 ID: 选择设备所属项目
   - 安装地址: 设备安装位置
   - GPS 坐标: 设备经纬度（用于 GIS 大屏）
3. **保存设备**: 点击「保存」按钮

### 3.3 设备状态

- **在线状态**: 设备正常连接，显示绿色
- **离线状态**: 设备断开连接，显示红色
- **未知状态**: 设备状态未知，显示灰色

### 3.4 设备数据同步

1. **选择设备**: 在设备列表中选择需要同步数据的设备
2. **点击同步按钮**: 点击「同步因子」按钮
3. **等待同步完成**: 系统会自动分析设备历史数据，同步因子信息

## 4. 数据监测

### 4.1 实时数据

- **查看实时数据**: 在左侧菜单中点击「数据监测」→「实时数据」
- **选择设备**: 选择需要查看的设备
- **查看因子**: 查看设备的各个监测因子数据
- **数据刷新**: 页面会自动刷新实时数据

### 4.2 历史数据

1. **选择设备**: 选择需要查看历史数据的设备
2. **设置时间范围**: 选择开始时间和结束时间
3. **选择因子**: 选择需要查看的监测因子
4. **查看数据**: 系统会显示选定时间范围内的历史数据
5. **导出数据**: 点击「导出」按钮导出数据为 Excel 文件

### 4.3 数据报表

- **日报表**: 查看每日数据汇总
- **周报表**: 查看每周数据汇总
- **月报表**: 查看每月数据汇总
- **自定义报表**: 设置自定义时间范围的报表

## 5. 告警管理

### 5.1 告警配置

1. **选择设备**: 在设备列表中选择需要配置告警的设备
2. **点击告警配置**: 点击「告警配置」按钮
3. **设置阈值**: 为每个监测因子设置告警阈值
4. **选择告警方式**: 启用短信告警、微信告警等
5. **保存配置**: 点击「保存」按钮

### 5.2 告警列表

- **查看告警**: 在左侧菜单中点击「告警管理」→「告警列表」
- **筛选告警**: 按设备、时间、级别等筛选告警
- **处理告警**: 点击告警记录进行处理
- **告警统计**: 查看告警统计信息

### 5.3 告警通知

- **短信通知**: 向指定手机号发送告警短信
- **微信通知**: 向指定微信用户发送告警通知
- **邮件通知**: 向指定邮箱发送告警邮件

## 9. 故障处理

### 9.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 设备离线 | 网络连接问题 | 检查设备网络连接，重启设备 |
| 数据异常 | 传感器故障 | 检查传感器状态，校准传感器 |
| 系统登录失败 | 账号密码错误 | 检查账号密码，联系管理员重置 |
| 告警不触发 | 阈值设置错误 | 检查告警阈值设置，重新配置 |
| 数据不更新 | 设备通信问题 | 检查设备通信状态，重启设备 |

### 9.2 故障排查

1. **检查设备状态**: 查看设备是否在线
2. **检查网络连接**: 检查设备网络连接是否正常
3. **查看系统日志**: 查看系统日志找出错误原因
4. **重启设备**: 尝试重启设备
5. **联系技术支持**: 如果问题无法解决，联系技术支持
