# TCD 串联参比气路系统

本章节详细展示了微型气相色谱 (Micro-GC) 中 TCD（热导池检测器）串联参比气路的物理工作过程。
重点演示了反吹进样时**“载气缓慢推着独立样气段（柱塞/Slug）进入色谱柱分离”**的严谨流体力学现象。

## 技术参数与注意事项

在设计色谱切换阀进样系统时，为了避免阀门切换瞬间因阻力突变导致 EPC 压力剧烈波动和基线脉冲（倒峰），
本设计在六通阀的旁路增加了一个与进样定量环气阻完全相等的空定量环（匹配阻尼管）。

- **定量环体积：** 1.0 mL
- **载气推荐压力：** 300 kPa
- **样气流速限制：** ≤ 50 mL/min，以防止背压过大。
- **切换时序：** 进样保持时间建议大于 15 秒，以确保样气完全进入色谱柱。
